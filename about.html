To play, you need to join the dvz.projectenyo.eu server when it is open. The server is running on the 1.19.3 version of Minecraft and is hosted in Europe, and games are run during EU friendly hours.
Even though the server accepts clients dating back to 1.7, it is recommended to play on a modern version.

Client Mods

If you are running a modern Minecraft version, there are some mods we recommend you to install.

  • Sodium
  • Starlight
  • No Chat Reports

Commands

Useful Commands
/compass Describes your current bearing.
/depth States current depth.
/trash Opens a portable disposal menu.
/helpop <message> Message online admins.
/ignore <player> Ignore other players.
/msg <player> Sends a private message to the specified player.
/msgtoggle Blocks receiving all private messages.
/r Quickly reply to the last player to message you.
/rtoggle Change whether the recipient of the reply is last recipient or last sender.
/time Display the world time.
/mana Shows your mana.
